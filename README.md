# Happy Paws Adoption Center 

**Happy Paws** is a multi-page static website designed for an animal rescue and adoption center. The project serves as a bridge between homeless pets and potential adopters, providing a clean and informative platform to showcase available animals, upcoming events, and ways for the community to get involved.

---

## Key Features

The website is structured into several specialized sections to ensure a great user experience:

* **Home Page:** Features a hero section, mission statement, video integration, business hours, and an "Upcoming Events" calendar.
* **About Us:** Details the organization's history (founded in 2018), their core mission, and introduces the professional team (Veterinarians, Coordinators, etc.).
* **Pets Gallery:** A catalog of animals currently available for adoption, including a detailed status table (Available vs. Adopted).
* **Volunteer & Donations:** Dedicated space for community support, explaining how to become a volunteer or contribute via monetary/item donations.
* **Contact & Location:** Provides specific contact channels for rescue reports or media requests, alongside a Google Maps integration for the Montreal-based center.

## Built With

* **HTML5:** Utilizing semantic tags (`<section>`, `<nav>`, `<header>`, `<footer>`) for better SEO and accessibility.
* **CSS3:** Custom styling for a cohesive brand identity, including card layouts and navigation.
* **YouTube API:** Embedded video content to showcase the center's impact.
* **Google Maps:** Interactive iframe for easy navigation to the physical location in Montreal.
